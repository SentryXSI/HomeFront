# HomeFront

Minimal localhost server frontend / admin control panel  

### WHAT ?

HomeFront is designed to be the homepage website for your web browser.  

#### Components 

Bridge  
Bookmarks  
Directory Index    
Download Manager  
Hosts File Manager   
Journal   
Notes    
Messages  
Password Vault      
Todo list      
Virtual Hosts Manager
